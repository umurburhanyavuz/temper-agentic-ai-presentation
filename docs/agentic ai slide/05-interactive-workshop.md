# Interactive Workshop Segment

Duration: ~8-10 min (embedded in the presentation, not a separate block)
Goal: Audience designs one workflow using the three pillars.

---

## Approach: Design a Workflow, Not Just a Gem

Instead of just building a Gem (which is only one piece), have the audience design a complete semi-automated workflow using all three pillars.

---

## The Worksheet (display on screen)

Ask everyone to fill this in mentally (or on paper / their phone):

```
MY WORKFLOW DESIGN

1. THE TASK
   What repetitive task do I do at least once a week?
   _______________________________________________

2. THE DATA (Pillar 1: Connections)
   What tools hold the data I need?
   [ ] Jira     [ ] Notion    [ ] Gmail     [ ] Slack
   [ ] Sheets   [ ] Mixpanel  [ ] Miro      [ ] Other: ____

3. THE FLOW (Pillar 2: Human in the Loop)
   Which steps can AI do automatically?
   Step 1 (auto): ________________________________
   Step 2 (auto): ________________________________
   Step 3 (HUMAN): _______________________________
   Step 4 (auto): ________________________________

4. THE CONTEXT (Pillar 3: Context Engineering)
   What does AI need to know to do this well?
   - My role: ____________________________________
   - Output format: ______________________________
   - Rules / constraints: ________________________
   - Team knowledge: _____________________________
```

---

## Role-Specific Workflow Ideas

| Role | Workflow | Data Sources | Human Gate |
|------|----------|-------------|------------|
| **PM** | Sprint summary to stakeholders | Jira + Notion | Review before sending |
| **Recruiter** | Candidate pipeline digest | ATS + Gmail | Personalize outreach |
| **Ops** | Shift fill rate weekly report | Mixpanel + Sheets | Verify anomalies |
| **Engineer** | PR description from code changes | Git + Jira | Review before submitting |
| **Designer** | Design review prep doc | Miro + Notion + Jira | Add subjective feedback |
| **Sales** | Client activity summary | CRM + Gmail + Sheets | Personalize before sending |
| **Manager** | 1:1 prep from team activity | Jira + Slack + Notion | Add personal topics |
| **Marketing** | Campaign performance brief | Mixpanel + Sheets | Interpret + decide next steps |
| **HR** | Onboarding checklist tracker | Notion + Gmail + Slack | Confirm completions |
| **Finance** | Expense anomaly report | Sheets + invoices | Approve/flag exceptions |

---

## Facilitation Script

### Introduce (0:00)
> "Now I want you to design your own workflow. Not just a prompt — a complete flow with data sources, AI steps, and a human checkpoint."

### Show the Worksheet (0:30)
> "Here's the template. Think about one task you do every week. What data do you need? Where does it live? What can AI do? Where should you stay in the loop?"

### Think Time (1:00 - 4:00)
> "Take 3 minutes. If you need inspiration, look at the role-specific ideas on screen."

### Share (4:00 - 8:00)
> "Who wants to share? Just tell us: what's the task, what tools are involved, and where's your human gate?"

Pick 3-4 people. For each:
1. Repeat their workflow back simply
2. Validate: "That's a great candidate for semi-automation"
3. If they missed a human gate: "Where would you want to review before it goes out?"
4. If they missed context: "What would AI need to know to get the output right?"

### Close (8:00)
> "You just designed an AI workflow in 3 minutes. The hard part isn't the AI — it's knowing your own process well enough to describe it. That's context engineering in action."

---

## If Time is Short

Skip the sharing segment. Instead:
> "I won't ask you to share now — but take this framework with you. This week, write down one workflow using this template. Post it in #ai-workflows. Let's collect the best ideas and start building them together."
