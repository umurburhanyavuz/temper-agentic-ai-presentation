## Key Concepts to Reference

### MCP in One Sentence
"An open protocol that lets any AI tool connect to any data source — like USB-C for AI."

### Context Engineering in One Sentence
"The discipline of structuring everything an AI needs — instructions, knowledge, memory, and tool access — so it consistently produces high-quality, relevant output."

### Human in the Loop in One Sentence
"AI handles the heavy lifting; humans approve at critical checkpoints — combining AI speed with human judgment."
